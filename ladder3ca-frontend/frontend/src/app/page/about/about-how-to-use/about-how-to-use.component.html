How to use

Annotation Assistant

Annotate a text

Enter a New Text:
Type the text (either a cancellation or a request) you wish to annotate into the provided field.

Select Language:
Choose the language of your text from the dropdown menu.

Press “Annotate“:
Click the "Annotate" button to proceed with the annotation.

Annotate Text with Advanced Features

Add a text

Find Texts

Enter a New Text:
Type your text (either a cancellation or a request).

Select Language:
Choose the language of your text.

Press “Annotate“:
Click the "Annotate" button.

Add as New Text:
Select this option to add your annotated text as a new entry.

Set Metadata:
Choose the language (Italian or German) and specify the L1/L2 language with the first three letters (e.g., Ita, Ger).

Incorporate into Corpus:
Your text will be added to the corpus and used in the training set, visible under “Last Changes.“

Find Text

Enter a Search Term:
Input your search term, such as “Hallo“ or “Intensifier“.

Confirm the Query:
Click the button under the field e.g. “Add Fulltext Clause” or “Add Modifier Clause“

Apply Filters:
Language selection is required. Apply other filters as needed.

Press “Search“:
Click the search button.

View Results Online:
Review the search results displayed.

Export Results:
Choose to export the results in JSON, TEI/XML, or CSV formats.

Modify Annotations

Access Modification Tools:
Click on the tag icon next to the message, situated between the pencil and bin icons.

Edit Annotation:
Click the pencil icon to modify the existing annotation.

Modify/Add Task

Create or Edit Tasks:
Click on the “Create Tasks“ button to add a new task or modify an existing one. Tasks are prompts that elicit specific responses or actions.

Modify/Add Modifiers and Subact Tag

Add or Modify Tags:
Click on the “Modifiers“ or “Subact“ button to add a new modifier or subact tag or to modify existing ones.

Modify/Add Models:

View and Edit Categories:
Click on a category tag to view or modify the classification categories. The system will train the model based on these categories, which automatically update once a day.
